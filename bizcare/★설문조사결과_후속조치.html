재방문을 유도할 수 있는 서비스를 여러 채널로 안내


1. 비즈케어 서비스 홍보사이트(블로그 등)

2. 검진안내 이메일에 3개년 누적관리 안내

3. 각 메뉴 접근 편의성 보완

설문결과

후속조치

검진예약 Form 편의성 확보


1. 행정구역 세분화 / 주소기반 병원 검색

2. 검진희망날짜로 검색

검진항목 보기, 비교하기 가독성 확보


1. 전체보기 / 간략히 보기로 구분

2. 선택검사항목 / 추가검사항목 구분하여 출력


병원정보내 사진 컨텐츠 추가


1. 병원관리자에 담당자가 직접 업로드

이용자 평가 기능 추가


1. 소항목 몇 가지로 이용자 별점 기능 추가(예. 교통편의성, 병원친절도, 시설만족도 등: 비즈케어팀 정리요망)


* 전문가평가는 객관성 판단의 근거가 모호하여 제외


2. 전체 검색 결과의 병원정보에 구성

기존 검진항목 검토


1. 기존 CM에 등록된 분류별 카테고리 및 항목명, 중복여부, 설명 검토하여 재정리(비즈케어팀 정리요망)



1. 중복항목 검토

2. 용어 표기 및 설명문구 검토

(1,2번 비즈케어팀 정리요망)


3. 선택검사만 보는 기능 추가

사이트맵 재구성


1. 전체 4개 대메뉴로 재그루핑

2. 이용빈도 고려 메인화면에 바로가기 구성

부가서비스 노출 비중 확대


1. 사이트내 메뉴 및 바로가기 링크

2. 수검 후 제공되는 부가서비스 안내(평가보고서, 질환관리 메일)

3. 비즈케어 소개 사이트에 제공 서비스 홍보

구분

서술문항 의견

개선방향

예약관련

• 검진예약일 변경이 불편함

• 예약 후 내역 조회 화면(마이페이지) 구성 개선

컨텐츠 관련

• 검진센터의 자세한 위치안내가 없어서 찾는데 어려움

• 자주 질문하는 FAQ 메뉴가 없어서 불편함

• 검진날 몇 시까지 와야하는지 안내가 없음

• 검사가 어떻게 진행이 되는지 검사방법이 없었고, 사전 주의사항에 대한 공지가 없어 불편함

• 병원정보 내 주소 및 지도 가독성 개선

• FAQ 내용 보완 및 메뉴 접근성 개선 (비즈케어팀)

• 병원정보내 검진시간 안내 (비즈케어팀)

• 검진 D-day 안내 이메일에 수검자 유의사항 보완(비즈케어팀)

사이트 UI

• 메뉴구성이 복잡하여 메뉴별 접근성이 어려움

• 예약절차에 대한 소개가 없어 제대로 예약하였는지 의문이 듦

• 글자 및 전체적인 메뉴크기가 작아서 가독성이 떨어짐

• 텍스트위주라 가독성이 떨어짐

• 전체 4개 대메뉴로 재그루핑

• 예약Form 전송 후 신청완료 메세지 가독성 증대

• 디자인 및 사이트 전체적인 구성 개선

기타

• 사이트 URL이 어려워 바로들어가기 힘들었음

• 대표 URL로 시스템 변경

구분

서술문항 의견

개선방향

예약관련

• 기본항목에 있는 검사를 추가검사로 또 신청하였을 때 자동으로 신청 못하게 하는 기능

• CM프로그램내 유사항목 카테고라이징 검토(비즈케어팀)

컨텐츠 관련

• 나이대별 꼭 필요한 검사에 대한 컨텐츠 추가

• 대장내시경 가능일자를 사이트상에 표시해주는 기능

• 해당 검진센터 의료진의 프로필

• 검진샘플(ex . 40세 남성 홍길동씨는 매주 2회이상 음주를 하고 하루 1갑정도의 담배를 피우다가 올해 건강검진에서 폐CT를 촬영하였는데.......)

• 검진센터의 위치정보가 네이버지도 등의 지도와 연계되는 기능

• 연령대별 추천 검사 컨텐츠 보완(비즈케어팀)

• 병원관리자에 기능 추가

• 의료진 프로필 정보 보완(비즈케어팀)

• 검진샘플(최은진대리 조사)

• 지도연계하여 표기

기타

• 검진센터 별 문진표를 사이트에서 다운받을수 있으면 검진시간이 줄어들 것임

• 현재 사는곳에서 가까운 위치의 검진센터를 조회하는 기능

• 병원별 문진표 양식 정리(비즈케어팀)하여 전달

• 고객 주소지 기반 검진센터 자동선택기능 추가
